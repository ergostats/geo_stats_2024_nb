Iniciamos con la instalación de R base, esta instalación basta para nuestros propósitos, sin embargo hoy en día tenemos a disposición el maravilloso IDE (integrated develpment environment) de RStudio que nos ayuda a que la curva de aprendizaje sea más ligera para los estudiantes.

Descarga

Este nos dará la descarga de un archivo .exe. Una vez descargado, empezamos la instalación:

  • Primero, debemos elegir el lenguaje de preferencia
  • Segundo, aceptamos los terminos y condiciones
  • Tercero, elegimos la ruta en la que se guardará el programa
  • Después, elegimos qué componentes deben ser instalados (se recomienda dejar todas las opciones activas)
  • Luego, elegimos aceptar todas estas configuraciones
  • A continuación, elegimos el tipo de display (se recomienda la establecida por default)
  • Enseguida, elegimos ayuda HTML para los textos auxiliares
  • Posteriormente, elegimos una ruta para las librerías. Si se desea una carpeta en el menú de inicio, se recomienda dejar las opciones por default
  • Finalmente, algunas alternativas adicionales como son la creación de una acceso rápido y la posibilidad de guardar un registro de varias versiones de R (en esta ultima se recomienda mantener un historial por motivos de reproducibilidad)

Con eso concluye el proceso de instalación esperamos unos minutos y tendremos listo nuestro programa.

Recomendaciones para trabajar con Rstudio

Uso de proyectos

Rstudio te permite convertir un directorio o carpeta de tu computador en un proyecto, esto será de gran ayuda para trabajar con archivos de manera centralziada, ordenada y eficiente. Cuando estableces un proyecto en Rstudio tienes la ventaja que tu carpeta se convertira en la “raiz” del proyecto, con ello será más facil construir rutas a tus archivos y scripts, ademas de que podrás trasladar tu trabajo a otro computador de manera más sencilla.

Remoción de restricciones para la instalación

Para poder instalar librerias de R sin problemas vamos a ir a la opción Tools > Global Options > Packages y retiramos el cajon marcado “Use securre download methos HTTP”:

Resaltar llamadas de funciones y paréntesis de colores

Para poder hacer del desarrollo de nuestros scripts de de R una tarea sencilla y ordenada vamos a aprovechar dos características que ofrece Rstudio:

  • Resaltar la llamada de funciones nos permitirá distinguir en el código las funciones de las variables y otras expresiones, lo cual hará más legible tu código y facilitará su interpretación cuando trabajes con tus colegas.

  • Los paréntesis de colores son especialmente útiles cuando anidamos funciones (usamos llamadas de función dentro de otra, y así sucesivamente).

Para activarlos nos vamos a la opción Tools > Global Options > Code> Display y señalamos los recuadros de la sección Syntax.

Documentos Quarto

Para el curso que vamos a desarrollar juntos. Quarto es una herramienta excelente que permite combinar código, análisis y narrativa en un solo documento, muy similar a lo que hacemos en Word.

¿Cómo empezar con Quarto en RStudio?

  1. Instalación y Configuración Inicial:
  • Primero, asegúrate de tener RStudio y Quarto instalados. Puedes instalar Quarto desde aquí.
  1. Crear un Nuevo Documento Quarto:
  • En RStudio, ve a File > New File > Quarto Document. Esto te creará una plantilla básica con la que puedes empezar a trabajar.
  • Si es la primera vez que abres un documento Quarto, RStudio te dará la opción de instalar la librería Quarto, caso contrario puedes seguir las instrucciones del paso 1.
  1. Combinar Código y Narrativa:
  • La magia de Quarto es que puedes escribir texto y agregar bloques de código en R, Python, o Julia.
  • Puedes escribir tu narrativa alrededor de bloques de código, explicando tus análisis y resultados.
  1. Modos de Edición: Visual y Source:

  • RStudio ofrece dos modos de edición para Quarto: Visual y Source.

  • Visual es más amigable y se siente como trabajar en Word. Aquí puedes ver cómo se verá tu documento final mientras lo editas.

  • Source es ideal si prefieres trabajar directamente con el código y el markdown.

  • Para cambiar entre estos modos, puedes usar los botones Visual y Source en la esquina superior derecha del panel de edición.
